# flask-template

A quickstart Flask template.

### Install

```sh
$ git clone https://github.com/remarkablemark/flask-template.git
$ cd flask-template
$ pip install -r requirements.txt # you may need `sudo`
```

### Run

```sh
$ FLASK_APP=app.py flask run
```
